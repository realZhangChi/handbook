        Rename branch #

        git branch -m NEW_BRANCH_NAME

        Ignore local file modifications #

        git update-index --no-skip-worktree /path/to/file

        Delete a Git branch locally #

        git branch --delete <branchname>

        See also How to delete a Git branch locally.

        Update submodules recursively #

        git submodule update --init --recursive

        See also Git update submodules recursively

        Config proxy #

        git config --global https.proxy http://127.0.0.1:7890

git config --global https.proxy https://127.0.0.1:7890

git config --global --unset http.proxy

git config --global --unset https.proxy

        See also git 设置和取消代理

        How to push empty commits to a remote repository #

        git commit --allow-empty -m "Trigger Build"

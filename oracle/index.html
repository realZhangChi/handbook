          Find duplicates #

          SELECT column_name, COUNT(column_name)
FROM table_name
GROUP BY column_name
HAVING COUNT(column_name) > 1;

          See also How do I find duplicate values in a table in Oracle?.

          Removing duplicates #

          DELETE FROM your_table
WHERE rowid not in
(SELECT MIN(rowid)
FROM your_table
GROUP BY column1, column2, column3);

          See also Removing duplicate rows from table in Oracle.

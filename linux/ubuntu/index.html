      Ubuntu
      • Install .deb
      • Set timezone
      • Change NTP server

      Install .deb #

      sudo dpkg -i package-name.deb

      Set timezone #

      timedatectl set-timezone Asia/Shanghai

      Change NTP server #

      1. Open and Edit the configuration file

        vim /etc/systemd/timesyncd.conf

      2. Add(Uncomment) NTP servers under the [Time] like following

        [Time]
NTP=10.0.0.1

      3. Restart the service to apply the changes

        systemctl restart systemd-timesyncd.service

      4. To check time sync status

        timedatectl status

      See also How to change NTP server on Ubuntu 20.04.1 LTS simply

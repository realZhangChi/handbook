    Extension #

    System.IO.Path.GetExtension #

    [System.IO.Path]::GetExtension(Draftv2.zip)

    Returns the extension (including the period “.”) of the specified path string. See also M$ Learn.

    FileSystemInfo.Extension #

    > (Get-ChildItem Draftv2.zip).Extension
.zip
> (Get-Item Draftv2.zip).Extension
.zip

    Gets the extension part of the file name, including the leading dot . even if it is the entire file name, or an empty string if no extension is present. See also M$ Learn.

    Expand-Archive #

    Expand-Archive -Path Draftv2.zip -DestinationPath C:\Reference

    The Expand-Archive cmdlet extracts files from a specified zipped archive file to a specified destination folder. See also M$ Learn.

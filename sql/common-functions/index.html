          COALESCE #

          SELECT COALESCE(NULL, NULL, NULL, 'W3Schools.com', NULL, 'Example.com');

          Return the first non-null value in a list. See also w3schools.

          IFNULL #

          SELECT IFNULL(NULL, "W3Schools.com");

          Return the specified value IF the expression is NULL, otherwise return the expression. See also w3schools.

          IF #

          SELECT IF(500<1000, "YES", "NO");

          The IF() function returns a value if a condition is TRUE, or another value if a condition is FALSE. See also w3schools.

    Usage #

OK
> INCR foo
QUEUED
> INCR bar
QUEUED
> EXEC
1) (integer) 1
2) (integer) 1

    DISCARD #

    > SET foo 1
OK
> MULTI
OK
> INCR foo
QUEUED
> DISCARD
OK
> GET foo
"1"

    Optimistic locking using check-and-set #

    WATCH mykey
val = GET mykey
val = val + 1
MULTI
SET mykey $val
EXEC

    It is also possible to use the UNWATCH command manually in order to flush all the watched keys.

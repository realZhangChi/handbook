    Database #

    DBSIZE #

    DBSIZE

    Return the number of keys in the currently-selected database. see also redis.io.

    FLUSHALL #

    
FLUSHALL [ASYNC | SYNC]

    Remove all keys from all database. see also redis.io.

    FLUSHDB #

    FLUSHDB [ASYNC | SYNC]

    Remove all keys from the current database. see also redis.io.

    SELECT #

    SELECT index

    Change the selected database for the current connection. see also redis.io.

    Key #

    KEYS #

    KEYS pattern

    Returns all keys matching pattern. See also redis.io.

    EXISTS #

    EXISTS key [key ...]

    Determine if a key exists. See also redis.io.

    EXPIRE #

    EXPIRE key seconds [NX | XX | GT | LT]

    Set a key’s time to live in seconds. See also redis.io.

    TTL #

    TTL key

    Get the time to live for a key in seconds. See also redis.io.

    TYPE #

    TYPE key

    Determine the type stored at key. See also redis.io.

    STRLEN #

    STRLEN key

    Get the length of the value stored in a key. See also redis.io.

    Value #

    GET #

    GET key

    Get the value of a key. See also redis.io.

    GETRANGE #

    GETRANGE key start end

    Get a substring of the string stored at a key. See also redis.io.

    APPEND #

    APPEND key value

    Append a value to a key. See also redis.io.

    SET #

    SET key value [NX | XX] [GET] [EX seconds | PX milliseconds |
  EXAT unix-time-seconds | PXAT unix-time-milliseconds | KEEPTTL]

    Set the string value of a key. See also redis.io.

    SETRANGE #

    SETRANGE key offset value

    Overwrite part of a string at key starting at the specified offset. See also redis.io.

    MGET #

    MGET key [key ...]

    Get the values of all given keys. See also redis.io.

    MSET #

    MSET key value [key value ...]

    Set multiple keys to multiple values. See also redis.io.

    MSETNX #

    MSETNX key value [key value ...]

    Set multiple keys to multiple values, only if none of the key exist. See also redis.io.

    INCR #

    INCR key

    Increment the integer value of a key by one. See also redis.io.

    DECR #

    DECR key

    Decrement the integer value of a key by one. See also redis.io.

    INCRBY #

    INCRBY key increment

    Increment the integer value of a key by the given number. See also redis.io.

    DECRBY #

    DECRBY key decrement

    Decrement the integer value of a key by the given number. See also redis.io.

    List #

    LPUSH #

    LPUSH key element [element ...]

    Prepend one or multiple elements to a list. See also redis.io.

    RPUSH #

    RPUSH key element [element ...]

    Apend one or multiple elements to a list. See also redis.io.

    LRANGE #

    LRANGE key start stop

    Get a range of elements from a list. See also redis.io.

    LINDEX #

    LINDEX key index

    Get an element from a list by its index. See also redis.io.

    LPOP #

    LPOP key [count]

    Remove and get the first elements in a list. See also redis.io.

    RPOP #

    RPOP key [count]

    Remove and get the last elements in a list. See also redis.io .

    LLEN #

    LLEN key

    Get the length of a list. See also redis.io.

    LREM #

    LREM key count element

    Remove elements of a list. See also redis.io.

    LTRIM #

    LTRIM key start stop

    Trim the list to the specified range. See also redis.io.

    LSET #

    LSET key index element

    Set the value of an element in a list by its index. See also redis.io.

    LINSERT #

    LINSERT key <BEFORE | AFTER> pivot element

    Insert an element before or after another element in list. See also redis.io.

    Set #

    SADD #

    SADD key member [member ...]

    Add one or more members to a set. See also redis.io.

    SMEMBERS #

    SMEMBERS key

    Get all the members in a set. See also redis.io.

    SISMEMBER #

    SISMEMBER key member

    Determine if a given value is a member of a set. See also redis.io.

    SCARD #

    SCARD key

    Get the number of members in a set. See also redis.io.

    SREM #

    SREM key member [member ...]

    Remove one or more members from a set. See also redis.io.

    SRANDMEMBER #

    SRANDMEMBER key [count]

    Get one or multiple random members from a set. See also redis.io.

    SPOP #

    SPOP key [count]

    Remove and return one or multiple random members from a set. See also redis.io.

    SMOVE #

    SMOVE source destination member

    Move a member from one set to another. See also redis.io.

    SDIFF #

    SDIFF key [key ...]

    Subtract multiple sets. See also redis.io.

    SINTER #

    SINTER key [key ...]

    Intersect multiple sets. See also redis.io.

    SUNION #

    SUNION key [key ...]

    Add multiple sets. See also redis.io.

    Hash #

    HGET #

    HGET key field

    Get the value of a hash field. See also redis.io.

    HSET #

    HSET key field value [field value ...]

    Set the string value of a hash field. See also redis.io.

    HMGET #

    HMGET key field [field ...]

    Get the values of all given hash fields. See also redis.io.

    HGETALL #

    HGETALL key

    Get all the fields and values in a hash. See also redis.io.

    HDEL #

    HDEL key field [field ...]

    Delete one or more hash fields. See also redis.io.

    HLEN #

    HLEN key

    Get the number of fields in a hash. See also redis.io.

    HEXISTS #

    HEXISTS key field

    Determine if a hash field exists. See also redis.io.

    HKEYS #

    HEXISTS key field

    Get all the fields in a hash. See also redis.io.

    HVALS #

    HVALS key

    Get all the values in a hash. See also redis.io.

    HINCRBY #

    HINCRBY key field increment

    Increment the integer value of a hash field by the given number. redis.io.

    HSETNX #

    HSETNX key field value

    Set the value of a hash field, only if the field does not exist. See also redis.io.

    Sorted Set #

    ZADD #

    ZADD key [NX | XX] [GT | LT] [CH] [INCR] score member [score member...]

    Add one or more members to a sorted set, or update its score if it already exists. See also redis.io.

    ZRANGE #

    ZRANGE key start stop [BYSCORE | BYLEX] [REV] [LIMIT offset count] [WITHSCORES]

    Return a range of members in a sorted set. See also redis.io.

    ZREM #

    ZREM key member [member ...]

    Remove one or more members from a sorted set. See also redis.io.

    ZCARD #

    ZCARD key

    Get the number of members in a sorted set. See also redis.io.

    ZCOUNT #

    ZCOUNT key min max

    Count the members in a sorted set with scores within the given values. See also redis.io.
